

# Standard Developer Environment

## IDEs

- Roblox Studio
- Superbulletai: https://docs.superbulletstudios.com/installation
- Visual Studio Code + Rojo plugin

## CLI tools

- Rokit: https://github.com/rojo-rbx/rokit
```
Invoke-RestMethod https://raw.githubusercontent.com/rojo-rbx/rokit/main/scripts/install.ps1 | Invoke-Expression
```
- Wally
```
rokit add wally
```
- Lune: https://lune-org.github.io/docs/getting-started/1-installation/

```
rokit add lune
```
- Jest-Lua: https://jsdotlua.github.io/jest-lua/
```
wally install
```
- Rojo: https://rojo.space/docs/v7/getting-started/installation/
